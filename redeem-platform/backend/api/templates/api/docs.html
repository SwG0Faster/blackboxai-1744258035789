CodeMarket API Documentation

API Status: Running

Available Endpoints

Authentication

POST /api/auth/login/

Login with username and password

POST /api/auth/register/

Register a new user account

POST /api/auth/logout/

Logout current user

Profile

GET /api/profile/

Get user profile information

Codes

GET POST /api/codes/

List and create redeem codes

Additional Endpoints:

  • /api/codes// - Get, update or delete specific code
  • /api/codes/?is_active=true - List active codes

Transactions

GET POST /api/transactions/

List and create transactions

For raw JSON format, send request with Accept: application/json header
